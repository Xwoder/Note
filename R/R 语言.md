## 数学运算法

| 运算符 | 含义 |   示例    | 示例结果 |
| :----: | :--: | :-------: | :------: |
|  `+`   |  加  |  `2 + 3`  |   `5`    |
|  `-`   |  减  |  `2 - 3`  |   `-1`   |
|  `*`   |  乘  |  `2 * 3`  |   `6`    |
|  `/`   |  除  |  `6 / 2`  |   `3`    |
|  `^`   |  幂  |   `2^3`   |   `8`    |
|  `%%`  |  模  | `12 %% 5` |   `2`    |

## 逻辑运算

| 运算符 |    含义    |   示例   | 示例结果 |
| :----: | :--------: | :------: | :------: |
|  `>`   |    大于    | `2 > 1`  |  `TRUE`  |
|  `>=`  | 大于或等于 | `2 >= 2` |  `TRUE`  |
|  `<`   |    小于    | `2 < 1`  | `FALSE`  |
|  `<=`  | 小于或等于 | `2 <= 2` | `FALSE`  |
|  `==`  | 等于 | `2 == 2` | `TRUE`  |
|  `!=`  | 不等于 | `2 != 2` | `FALSE`  |
