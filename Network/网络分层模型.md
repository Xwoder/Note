# 网络分层模型

## OSI 模型

1. 应用层 / Application Layer
2. 表示层 / Presentation Layer
3. 会话层 / Session Layer
4. 传输层 / Transport Layer
5. 网络层 / Network Layer
6. 数据链路层 / Data Link Layer
7. 物理层 / Physical Layer

## TCP/IP 模型

1. 应用层 / Application Layer
2. 传输层 / Transport Layer
3. 网际层
4. 网络接口层

## 五层模型

1. 应用层 / Application Layer
2. 传输层 / Transport Layer
3. 网络层
4. 数据链路层 / Data Link Layer
5. 物理层 / Physical Layer
