# JavaScript 数据类型

## 分类

- 原始类型
    - `number`
    - `string`
    - `boolean`
    - `null`
    - `undefined`

- 引用类型

使用 `typeof` 关键字可以查看变量或常量以及字面量的类型

### `number`

> 数值类型

其中包含一个特殊的值：`NaN`（`Not a Number`）

### `string`

> 字符串类型

通过使用单引号或双引号包围来定义

```javascript
'Hello World'
"Hello JavaScript"
```

### `boolean`

> 布尔类型

只有两种值：`true` 或 `false`

### `null`

> 空类型

### `undefined`

> 未定义类型
