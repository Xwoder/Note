# JavaScript 引入

## 内部脚本

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <script>
        function sayHello() {
            console.log("Hello World");
        }

        sayHello();
    </script>
    <title>Title</title>
</head>
<body>

</body>
</html>

<script>
    function sayHi() {
        console.log("Hi, JavaScript");
    }

    sayHi();
</script>
```

## 外部脚本

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <script src="https://cdn.bootcdn.net/ajax/libs/jquery/3.6.0/jquery.js"></script>
    <script src="https://cdn.bootcdn.net/ajax/libs/vue/3.2.36/vue.cjs.js"></script>
    </script>
    <title>Title</title>
</head>
<body>

</body>
</html>
```
