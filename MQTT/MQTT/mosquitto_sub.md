# Eclipse Mosquitto mosquitto_sub

[TOC]

用于订阅主题的客户端，支持 `MQTT 5`、`MQTT 3.1.1` 和 `MQTT 3.1`

## 选项

### 主题 / `topic`

> 主题

|   全写    | 缩写 | 默认值 |
| :-------: | :--: | :----: |
| `--topic` | `-t` |   无   |

缩写：`-t`

### 主机 / host

> 主机

|   全写   | 缩写 |   默认值    |
| :------: | :--: | :---------: |
| `--host` | `-h` | `localhost` |

### 端口 / port

> 端口

|   全写   | 缩写 |      默认值      |
| :------: | :--: | :--------------: |
| `--port` | `-p` | `1883` 或 `8883` |

### 服务质量 / ` QoS`

> 服务质量

|  全写   | 缩写 | 默认值 |
| :-----: | :--: | :----: |
| `--qos` | `-q` |  `0`   |

### 调试 / ` debug`

> 输出调试信息

|   全写    | 缩写 | 默认值 |
| :-------: | :--: | :----: |
| `--debug` | `-d` | 无参数 |

## 综合示例

```bash
$ mosquitto_sub \
    --debug \
    --host "127.0.0.1" \
    --port 1883 \
    --id "Device_B" \
    --topic "sensor/temperature/#"
```
