## 命令

### list

```bash
$ brew list
```

显示所有安装的 `formula`

### leaves

```bash
$ brew leaves
```

显示所有安装的且不依赖其他 `formula` 的 `formula`

### cask

#### list

```bash
$ brew cask list
```
